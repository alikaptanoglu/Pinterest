Mock Pinterest Website
===
This is a website that is to recreate some of the main functions of the Pinterest website using HTML, CSS, JavaScript, JQuery, PHP, MySQL, and Bootstrap.

**Features to be replicated:**
- Creating an account
- Logging in
- Creating boards
- Pinning items
